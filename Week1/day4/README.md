---

# 📖 What is Flow Builder?
Flow Builder is a Salesforce automation tool used to automate business processes without writing code.

It helps businesses automate repetitive tasks, update records automatically, send notifications, and improve workflow efficiency.

---

# Types of Flows

## Screen Flow
A Screen Flow interacts with users using screens and input forms.

Example:
- Student registration form

---

## Record Triggered Flow
A Record Triggered Flow runs automatically when a record is created, updated, or deleted.

Example:
- Automatically updating remaining course seats after student registration

---

# 💡 Automation Ideas – College Management System

## 1. Auto Email After Registration
Automatically send confirmation email after student registration.

## 2. Auto Update Remaining Seats
Automatically reduce available seats when a student enrolls.

## 3. Notify Faculty When Course is Full
Send notification to faculty when course seats are filled.

## 4. Generate Student ID Automatically
Automatically create unique student IDs.

## 5. Fee Deadline Reminder
Send reminder email before fee payment deadline.

# 🔄 Flow Design – Student Registration Automation

## Trigger
Student registration form submitted.

## Steps
1. Create student record
2. Generate student ID
3. Update remaining seats
4. Send confirmation email

## Decision Point
Check if seats are available.

## Final Action
Student successfully registered.

---

# ⚙️ Manual vs Automated Process

## Manual Process
Staff manually enters student details, updates seats, and sends emails.

### Problems
- Time consuming
- Human errors
- Delayed updates
- Repetitive work

---

## Automated Process
Salesforce automatically updates records, sends emails, and manages workflows.

### Benefits
- Faster process
- Better accuracy
- Improved productivity
- Consistent workflow
