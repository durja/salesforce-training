# 🚀 Salesforce Summer Program – Day 5 Apex Introduction

---

# 📖 What is Apex?
Apex is a Salesforce programming language used for advanced business logic and automation.

---

# 🔄 Flow vs Apex

## Flow
- No-code automation
- Used for simple tasks

Examples:
- Auto emails
- Record updates

## Apex
- Coding-based solution
- Used for complex logic

Examples:
- Fee calculations
- External integrations

---

# ⚙️ Configuration vs Coding

## Configuration
Customization using clicks.

Examples:
- Validation Rules
- Flows
- Formula Fields

## Coding
Customization using Apex code.

Examples:
- Advanced automation
- API integrations

---
